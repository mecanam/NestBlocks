ブラウザ上で動作するマイコン用ビジュアルプログラミングエディタです。
> GithubPagesURL : https://mecanam.github.io/NestBlocks/
# Requirement
  OS : Windows , Mac (iOS未対応)  
  Browser : MicrosoftEdge , GoogleChrome

# Installation
任意のディレクトリ内に移動して、以下のコマンドをシェルで実行してください。
```bash
git clone https://github.com/mecanam/NestBlocks.git
```
# Usage
index.htmlをブラウザ(MicrosoftEdge or GoogleChrome)で実行して下さい。  
> **WebSerial未対応ブラウザではマイコンに接続ができません。**  

> マイプロジェクト機能にて、***IndexedDB***を使用しています。そのため、ブラウザ終了時にキャッシュが自動的にクリアされる設定の場合、**データが消えてしまいます。** ブラウザ終了時に **キャッシュクリアする設定を解除する** ことを推奨します。
